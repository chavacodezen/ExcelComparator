# Comparador de Archivos Excel

Este proyecto permite comparar dos archivos Excel y transferir datos de una columna en el archivo fuente a otra columna en el archivo principal, basado en la coincidencia de valores en dos columnas específicas. El proyecto está diseñado con una interfaz gráfica en Python, lo que facilita su uso sin necesidad de modificar código.

## Requisitos

Antes de ejecutar el programa, no tengas en abiertos los archivos que seleccionaras para procesar.
